# 群组（Category/Board）管理 Markdown 导出功能 — 技术调研与实现方案

> **调研时间**：2026-07-01  
> **调研方法**：代码库逆向分析（前端 + 后端 + 全链路）  
> **项目**：Focalboard（Mattermost 社区分支）  
> **调研目标**：梳理群组管理全链路，为"群组选项菜单添加 Markdown 导出"功能提供完整实现方案

---

## 1. 研究概述

### 1.1 需求理解

用户希望在群组（Category / Board）的选项菜单中添加一个 Markdown 导出功能，入口为"群组选项菜单中的下拉菜单"。

### 1.2 术语澄清

Focalboard 中没有传统意义上的"群组"（Group）概念。代码库中与"群组"最接近的概念是：

| 概念 | 代码中的名称 | 说明 |
|------|-------------|------|
| 分类/分组 | **Category** | 侧边栏中用于组织 Board 的文件夹，用户可自定义 |
| 看板 | **Board** | 项目看板，包含卡片（Card）和视图（View） |
| 卡片 | **Card** | 看板中的任务卡片，是 Block 的一种 |
| 分组属性 | **Group By** | 按属性值对卡片进行分组展示（Kanban/Table 视图） |

> **结论**：用户说的"群组"实际上是指侧边栏中的 **Category**（分类）及其下的 **Board**（看板）。导出操作应面向 Board 级别。

class CsvExporter {
    static exportTableCsv(board, activeView, cards, intl) {
        // 1. 生成 CSV 字符串
        const rows = this.generateTableArray(board, cards, viewToExport, intl)
        let csvContent = 'data:text/csv;charset=utf-8,'
        rows.forEach(row => { csvContent += row.join(',') + '\r\n' })

        // 2. 创建下载链接
        const link = document.createElement('a')
        link.setAttribute('href', encodeURI(csvContent))
        link.setAttribute('download', `${filename}.csv`)
        document.body.appendChild(link)
        link.click()
    }
}

## 5. Markdown 导出实现方案

### 5.1 方案选择：纯前端导出

**推荐方案**：纯前端 Markdown 导出，不新增后端 API。

**理由**：
1. CSV 导出已经证明纯前端导出模式可行
2. Markdown 生成逻辑简单，不需要后端处理
3. 数据已在 Redux store 中（boards, cards, blocks, views）
4. 减少后端改动，降低维护成本
5. 即时响应，无需等待后端生成

### 5.2 涉及文件清单

| 文件 | 操作 | 说明 |
|------|------|------|
| `webapp/src/markdownExporter.ts` | **新建** | Markdown 导出器类 |

| `webapp/src/components/viewHeader/viewHeaderActionsMenu.tsx` | **修改** | 视图头部菜单添加 "Export to Markdown" |
| `webapp/src/components/sidebar/sidebarCategory.tsx` | **可选修改** | 分类菜单添加 "Export category to Markdown" |

### 5.3 实现步骤

#### 步骤 1：创建 Markdown 导出器

**文件**：`webapp/src/markdownExporter.ts`

```typescript
// 数据结构参考 csvExporter.ts 和 archiver.ts
import {IntlShape} from 'react-intl'
import {BoardView} from './blocks/boardView'
import {Board, IPropertyTemplate} from './blocks/board'
import {Card} from './blocks/card'
import {Utils} from './utils'
import {IAppWindow} from './types'
import propsRegistry from './properties'

declare let window: IAppWindow

class MarkdownExporter {
    /**
     * 导出看板为 Markdown 格式
     * @param board 看板对象
     * @param activeView 当前视图
     * @param cards 当前视图的卡片列表
     * @param intl 国际化对象
     */
    static exportMarkdown(board: Board, activeView: BoardView, cards: Card[], intl: IntlShape): void {
        const md = this.generateMarkdown(board, activeView, cards, intl)

        const filename = `${Utils.sanitizeFilename(board.title || 'Untitled')}.md`
        const blob = new Blob([md], {type: 'text/markdown;charset=utf-8'})
        const link = document.createElement('a')
        link.style.display = 'none'
        link.setAttribute('href', URL.createObjectURL(blob))
        link.setAttribute('download', filename)
        document.body.appendChild(link)
        link.click()
    }

    /**
     * 生成 Markdown 内容
     */
    private static generateMarkdown(board: Board, activeView: BoardView, cards: Card[], intl: IntlShape): string {
        const lines: string[] = []

        // 看板标题
        lines.push(`# ${board.icon || ''} ${board.title || 'Untitled Board'}`)
        lines.push('')

        // 看板描述
        if (board.description) {
            lines.push(board.description)
            lines.push('')
        }

        // 视图信息
        if (activeView) {
            lines.push(`> **View**: ${activeView.title || 'Untitled View'} | **Type**: ${activeView.fields.viewType}`)
            lines.push(`> **Cards**: ${cards.length}`)
            lines.push('')
        }

        // 属性定义
        if (board.cardProperties && board.cardProperties.length > 0) {
            lines.push('## Properties')
            lines.push('')
            lines.push('| Property | Type | Options |')
            lines.push('|----------|------|---------|')
            for (const prop of board.cardProperties) {
                const options = prop.options?.map(o => o.value).join(', ') || '-'
                lines.push(`| ${prop.name} | ${prop.type} | ${options} |`)
            }
            lines.push('')
        }

        // 卡片列表
        lines.push('## Cards')
        lines.push('')

        const visibleProperties = board.cardProperties.filter(
            (template: IPropertyTemplate) =>
                activeView?.fields.visiblePropertyIds?.includes(template.id)
        )

        if (activeView?.fields.viewType === 'table') {
            // 表格视图：输出 Markdown 表格
            this.generateTableMarkdown(lines, cards, visibleProperties, intl)
        } else {
            // 看板/日历/画廊视图：输出卡片列表
            this.generateCardListMarkdown(lines, cards, visibleProperties, intl)
        }

        // 导出信息
        lines.push('')
        lines.push('---')
        lines.push(`*Exported from Focalboard on ${new Date().toISOString().split('T')[0]}*`)

        return lines.join('\n')
    }

    /**
     * 生成表格格式 Markdown
     */
    private static generateTableMarkdown(
        lines: string[],
        cards: Card[],
        visibleProperties: IPropertyTemplate[],
        intl: IntlShape
    ): void {
        // 表头
        const headers = ['Name', ...visibleProperties.map(p => p.name)]
        lines.push('| ' + headers.join(' | ') + ' |')
        lines.push('| ' + headers.map(() => '---').join(' | ') + ' |')

        // 数据行
        for (const card of cards) {
            const row = [this.escapeMarkdown(card.title)]
            for (const template of visibleProperties) {
                let value = card.fields.properties[template.id]
                const property = propsRegistry.get(template.type)
                if (property.type === 'createdBy') value = card.createdBy
                if (property.type === 'updatedBy') value = card.modifiedBy
                row.push(this.escapeMarkdown(property.exportValue(value, card, template, intl) || ''))
            }
            lines.push('| ' + row.join(' | ') + ' |')
        }
    }

    /**
     * 生成卡片列表格式 Markdown
     */
    private static generateCardListMarkdown(
        lines: string[],
        cards: Card[],
        visibleProperties: IPropertyTemplate[],
        intl: IntlShape
    ): void {
        for (const card of cards) {
            lines.push(`### ${card.fields.icon || ''} ${this.escapeMarkdown(card.title)}`)
            lines.push('')
            for (const template of visibleProperties) {
                let value = card.fields.properties[template.id]
                const property = propsRegistry.get(template.type)
                if (property.type === 'createdBy') value = card.createdBy
                if (property.type === 'updatedBy') value = card.modifiedBy
                const displayValue = property.exportValue(value, card, template, intl)
                if (displayValue) {
                    lines.push(`- **${template.name}**: ${this.escapeMarkdown(displayValue)}`)
                }
            }
            lines.push('')
        }
    }

    /**
     * 转义 Markdown 特殊字符
     */
    private static escapeMarkdown(text: string): string {
        return text.replace(/([|\\`*_{}[\]()#+\-.!])/g, '\\$1')
    }
}

export {MarkdownExporter}
```
